# Using the Must Use Plugin

This document describes how to use the generic Must Use plugin as a storage location for the key used to encrypt and decrypt text with _Crypt for WordPress_.

## Notes

The "Must Use"-plugin file is automatically generated by _Crypt for WordPress_. This method is used automatically if the wp-config.php file is not writable. However, it can also be forced (see below).

## Usage

1. In the configuration for loading _Crypt for WordPress_ via `set_config()`, set the following parameters:
- “force_place” => “muplugin”
2. Save the settings. They take effect immediately.

## Options

Additional options include:

* “file_permissions” => Set the desired file permissions for the generated file, e.g., 0600 so that it is readable and writable only by the current server user
